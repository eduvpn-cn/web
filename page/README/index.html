uVPN 服务介绍

eduVPN 从 2010 年开始服务，eduVPN 目前提供以下服务：

  • openVPN (IPv6 only)
  • shadowsocks (IPv6 / IPv4)
  • openconnect / anyconnect (IPv4 only, by ocserv)

# 致谢

我们使用了以下开源软件 (项目) 来为您提供服务，包括但不限于：

  • CentOS ：基础系统。
  • ocserv：提供 openconnect/anyconnect 兼容 VPN 接入。
  • openVPN：提供 openvpn 接入。
  • shadowsocks：提供 shadowsocks 接入。
  • munin：用于系统监控。
  • fail2ban：用于系统加固。
  • freeradius：用于提供中心认证系统。
